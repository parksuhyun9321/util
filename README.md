# util

- 내가 썼던 라이브러리, 명령어

- -g : 전역으로 설치

- --save :  package.json의 dependencies에 모듈을 추가한다는 의미. 서버에서 실행될 때도 필요 (개발+프로덕션 환경. npm5 버전 이후에는 옵션을 쓰지 않아도 자동으로 추가)

- d : --save-dev 개발 환경에서만 설치(배포 시에는 설치 X)

- @latest : 최신 버전으로 설치

- npm i , pnpm add
